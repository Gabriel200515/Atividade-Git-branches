Fazer o merge entre as branches via GitHub. Explique os passos utilizados para fazer o merge. Houve conflito? Se sim, como você gerenciou isso?

R: Para poder mesclar (fazer o merge) e o pull request: 
>Clicar em "Compare & Pull Request";
>O GitHub verifica se é possível fazer  merge ou não, para mim, funcionou automaticamente;
>Para finalizar basta clicar em "Create pull request".
>O pull request será feito para o branch master, depois é só acessar master e aceitar o pull request.
